Q: Do I have to use React now?
A: For this project structure, yes for UI. The frontend is React.

Q: Can I switch between them?
A: React runs on localhost:5173 and uses PHP backend API on localhost/KvalDarbs/api.php.

Q: What about my database?
A: No changes! Both use the same database.

Q: Can I modify the React components?
A: Yes! Edit files in Frontend/src/ and changes appear instantly.

Q: How do I deploy?

Q: Can I add more features?

📞 NEED HELP?
═══════════════════════════════════════════════════════════════

If something doesn't work:

1. Check XAMPP is running (Apache + MySQL)
2. Check you're in the Frontend/ folder
3. Check Node.js is installed (node -v)
4. Check npm install ran successfully
5. Look at browser console (F12 → Console tab)
6. Check the documentation files listed above

Most issues are simple setup problems - easily fixed!
