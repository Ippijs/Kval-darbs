Welcome to the Hybrid App!

This project combines React frontend with PHP backend for a modern shopping experience.

Available Versions:

React App (Modern):
The React frontend is ready in the react-app/ folder.

cd react-app && npm install && npm run dev

Or visit the original PHP app:

Go to: /Frontend/home.php

Next Steps:

  1. Install Node.js and npm if you haven't already
  2. Navigate to the react-app folder
  3. Run: npm install
  4. Run: npm run dev
  5. Open http://localhost:5173 in your browser
